BeePlus for Beekeeper

Productivity add-ons for Beekeeper. Toggle features below.

How to use this extension

  1. Open Beekeeper in Chrome and log in.
  2. Hover over any profile picture (avatar).
  3. Come back here. The fields list will be populated automatically.
  4. Configure each feature. Changes save automatically.

Features

Privacy: profile data stays in your browser.
